設定印表機/掃描器

設定可從本應用程式列印的印表機和可從本應用程式接收掃描資料的掃描器。

搜尋印表機/掃描器

  1. 輕按主畫面的"設定"功能表。
  2. 輕按"印表機/掃描器設定"。
  3. 輕按"搜尋印表機/掃描器"，會自動搜尋出網路中的印表機和掃描器。
    注意：印表機/掃描器必須與行動裝置連接在同一網段內。若使用處於不同網段中的印表機/掃描器，請參閱    手動新增印表機/掃描器。
  4. 從網路中搜尋到的印表機和掃描器會顯示在清單內。
    顯示搜尋印表機/掃描器所搜尋到的"產品名稱"和"位置"。（"產品名稱（位置）"或"產品名稱（IP 位址）"顯示在"IP 位址"上方。）
    注意：在清單中顯示（"產品名稱（位置）"或"產品名稱（IP 位址）"）的格式可以在應用程式設定中設定。搜尋印表機/掃描器所搜尋到的"產品名稱"中包含的"SHARP"字樣，會忽略不顯示。

手動新增印表機/掃描器

  1. 輕按主畫面的"設定"功能表。
  2. 輕按"印表機/掃描器設定"。
  3. 若您欲新增任一印表機/掃描器與您的行動裝置在不同網段時(無法透過搜尋印表機/掃描器新增)，則請按一下"手動新增"。
  4. 下列項目可以設定在“印表機/掃描器資訊”和“操作設定”中設定。
    輕按"儲存"來儲存此設定。若你想欲取消新增，請在輕按"儲存"之前按螢幕外面的返回硬體鍵。
    名稱
    輸入您欲新增的印表機/掃描器名稱。若沒有輸入名稱，則會以裝置的IP位址當作為名稱。
    IP 位址
    輸入印表機/掃描器的IP位址。這是必要的欄位。
    注意：IP 位址已經手動新增則無法再輸入。
    連接埠號
    輸入印表機/掃描器的連接埠號。這是必要的欄位。
    注意：要從本應用程式列印則必須設定連接埠號。輸入與印表機設定相同的連接埠號。若輸入不同的連接埠號，則本應用程式無法列印到印表機。
    請檢查印表機的連接埠號，
    若在應用程式設定中沒有勾選"使用Raw列印"：請檢查印表機網頁上的[應用程式設定]-[PC列印設定]設定。印表機和本應用程式預設連接埠號為21。
    若勾選"使用Raw列印"：請檢查印表機網頁上的[網路設定]-[列印埠設定]-[RAW]選項的設定。預設連接埠號為9100。
    設定成預設的印表機/掃描器
    若選擇"I(開啟)"，則當你使用"列印"或"掃瞄"時，預設印表機/掃描器會設定此台印表機/掃瞄器。
    注意：若在應用程式設定勾選"自動選擇印表機/掃描器"，則無法設定"設定成預設的印表機/掃描器"。

檢視和編輯印表機/掃描器資訊

  1. 輕按主畫面的"設定"功能表。
  2. 輕按"印表機/掃描器設定"。
  3. 登錄的印表機和掃描器會顯示在清單內。
    此圖示是表示為自動新增的印表機/掃描器。
    此圖示是表示為手動新增的印表機/掃描器。
  4. 輕按指定的印表機/掃描器可檢視資訊。
  5. 將顯示所選印表機/掃描器的資訊。顯示在這裡的項目可用於手動新增印表機/掃描器。
    注意：自動新增的印表機/掃描器，IP 位址無法編輯。
    手動新增的印表機/掃描器，產品名稱和位置則無法編輯。當應用程式與印表機/掃描器初次連線時，產品名稱和位置欄位會自動填入。
  6. 當可以使用“從您的行動裝置掃描”時，則下列額外的設定可以在“操作設定”裡設定。
    從您的行動裝置掃描
    當啟用（選擇“I（開啟）”）時，設定掃描設定並從您的行動裝置掃描。
    顯示名稱
    在掃描器操作面板和您的行動裝置同時會顯示顯示名稱和驗證碼，以確保掃描到正確的目的掃描器。
    自動產生驗證碼
    當啟用時（選擇“I(（開啟）”），自動產生驗證碼。若禁用（選擇“O（關閉）“時，您必須在驗證碼欄位中輸入一組碼。
    驗證碼
    在掃描器操作面板和您的行動裝置同時會顯示驗證碼和顯示名稱，以確保掃描到正確的目的掃描器。
    重新設定掃描參數
    輕按此處會重新設定上一次操作掃描的掃描設定。
  7. 輕按"儲存"儲存設定。

移動或刪除清單中的印表機/掃描器

  1. 輕按主畫面的"設定"功能表。
  2. 輕按"印表機/掃描器設定"。
  3. 登錄的印表機和掃描器會顯示在清單內。
  4. 輕按"編輯"按鈕。
  5. 若您欲從清單中刪除印表機/掃描器，則可輕按然後"刪除"。
  6. 若您欲重新編排印表機/掃描器的順序則可輕按並透過手指上下移動。
    注意：刪除的印表機/掃描器能夠使用    搜尋印表機/掃描器再次新增或手動新增印表機/掃描器。

使用者資訊設定

設定使用者資訊。
  1. 輕按主畫面的"設定"功能表。
  2. 輕按"使用者資訊設定"。
  3. "程序定義資訊"是為了在"掃描"操作時登錄掃描目的地所需的設定。可用下列設定：
    顯示名稱
    在掃描器位址目錄中的掃描目的地。預設值與行動裝置設定中關於本機所設定的"名稱"相同。
    檢索字母
    檢索字母是用於掃描器位址目錄的螢幕中顯示"顯示名稱"的排序分類鍵。
  4. 用戶驗證是當印表機啟用用戶驗證時，欲列印到印表機所需的設定。可用下列的設定：
    使用登入名稱來驗證
    當印表機啟用使用登入名稱來驗證則選擇此處。輸入“登入名稱”和“密碼”。
    登入名稱
    輸入在這台印表機用戶驗證的登入名稱。
    密碼
    輸入在這台印表機用戶驗證的密碼。
    使用用戶號碼來驗證
    當印表機啟用使用用戶號碼來驗證則選擇此處。輸入“用戶號碼”。
    用戶號碼
    輸入在這台印表機用戶驗證的用戶號碼。
  5. 當選擇“預設工作ID”時，在操作面板上會顯示“使用者名稱”和“工作名稱”的設定。
    使用者名稱
    輸入使用者名稱將會顯示於印表機操作面板。若沒有設定，則會使用預設值"SharpdeskM User"。
    工作名稱
    輸入工作名稱將會顯示於印表機操作面板。若沒有設定，則印表機會使用檔案名稱來列印。
  6. 輕按"儲存"儲存設定。

應用程式設定

設定應用程式的操作。
  1. 輕按主畫面的"設定"功能表。
  2. 輕按"應用程式設定"。
  3. 以下可以在"操作設定"中設定。
    自動刪除程序定義
    若掃描完成後要刪除掃描器上的目的地，請選擇為"I(開啟)"。
    自動更新程序定義
    選擇為"I(開啟)"則不需要確認登錄，直接覆蓋上一次的掃描目的地。
    儲存匯入檔案
    若您欲儲存從其他應用程式接收的檔案以便讓本應用程式來列印，請選擇為"I(開啟)"。
    自動選擇印表機/掃描器
    選擇為"I(開啟)"，則根據連接到的無線網路(SSID)來自動選擇預設印表機/掃描器。
    高品質列印
    若您欲列印較高品質則選擇為"I(開啟)"。
    使用Raw列印
    選擇為"I(開啟)"則使用Raw連接埠（預設連接埠號為9100）則，而不是使用FTP連接埠（預設連接埠號為21）來列印。
    自動新增印表機/掃描器所預設的名稱。
    自動新增印表機/掃描器可以設定顯示的格式。可以選擇"產品名稱（位置）"或"產品名稱（IP 位址）"。
  4. "SNMP設定"是當若您的行動裝置連接到網路時，是使用非預設的SNMP GET群體名稱則必須設定。若您在這樣的網路環境中使用本應用程式，則請將"使用Public名稱搜尋"設定為"O(關閉)"，並在"輸入群體名稱"中填入正確的群體名稱。最多可以設定10個名稱。有關於SNMP GET的群體名稱，請洽您的網路管理員。
  5. 輕按"儲存"儲存設定。

電子郵件設定

設定您的電子郵件帳號讓此應用程式列印接收的電子郵件。
  1. 輕按主畫面的"設定"功能表。
  2. 輕按“電子郵件設定”。
  3. 以下為設定可用的電子郵件帳號：
    郵件帳號名稱
    輸入用戶名稱登錄到您的電子郵件帳號。
    此欄位當密碼設定時則是必須輸入。
    密碼
    輸入密碼登錄到您的電子郵件帳號。
  4. 以下為設定可用的電子郵件伺服器設定。請聯絡您的電子郵件管理員取得伺服器資訊。
    注意：電子郵件伺服器必須支援IMAP4。
    主機名稱
    輸入電子郵件伺服器主機名稱。
    連接埠號
    輸入電子郵件伺服器連接埠號。這是必填的欄位。若不勾選“SSL” 則預設連接埠號為143，或勾選則為993。
    SSL
    檢查是否需要使用SSL連接行動裝置和電子郵件伺服器。
  5. 輕按"儲存"儲存設定。

掃描

本功能為接收並儲存掃描的文件。
  1. 輕按主畫面的"掃描"。
  2. 輕按"掃描器:(掃描器名稱)"(在iPad，輕按下方的"掃描")選擇指定的掃描器。
    此圖示是指從印表機/掃描器清單中指定預設的掃描器。
操作方式取決於掃描器。
請參閱從您的行動裝置掃描學習如何從您的行動裝置執行掃描。若掃描設定功能無法在您的行動裝置中使用，請參閱從掃描器操作面板掃描。

從您的行動裝置掃描

請在您的行動裝置完成以下步驟來配置掃描設定。請前往應用程式的網站查詢支援的掃描器列表。
除了“在掃描器擺放文件”之外，所有的操作都可以在應用程式中完成。
注意：請確掃描器顯示螢幕的畫面為主畫面或用戶驗證畫面。若啟動外部帳號驗證，將顯示用戶驗證畫面。驗證成功後，將顯示掃描器的主畫面。
  1. 若您選擇支援從您的行動裝置掃描功能的掃描器，將顯示設定功能表和現在的掃描設定。
    注意：當禁用此功能時，您可以啟動從掃描器掃描。請參閱從掃描器操作面板掃描 取得進一步說明。
    可使用以下設定。若您是使用iPhone，透過輕按“完成”來儲存設定。若您使用iPad，則設定會自動儲存。
    色彩模式
    設定掃描原稿色彩（自動/全彩/灰階/黑白2階）。預設值為“自動”。
    注意：若在“原稿”中選擇“原稿規格”為“較長尺寸”時，只支援“黑白2階”。
    原稿
    設定欲掃描的原稿文件。可設定以下的項目：
    原稿規格
    當預設為“自動”時，掃描器無法偵測非標準原稿紙張規格。您必須要手動變更原稿規格設定。
    偵測到的紙張規格（僅顯示）
    若您在目標的掃描器設定文件，會自動偵測原稿紙張尺寸並且同時顯示在您的行動裝置和掃描器操作面板上。
    儲存尺寸
    可以手動設定掃描影像的尺寸。若手動設定的尺寸與偵測到紙張規格不同，影像將自動調整縮放成適合頁面。預設值為“自動”。
    注意：當選擇以下任何原稿紙張規格時，您只能選擇”自動“：較長尺寸/日本明信片/名片/2L尺寸/卡片/自訂紙張尺寸。
    登錄自訂紙張尺寸
    可以登錄自訂紙張尺寸。最多可以登錄5個自訂紙張尺寸。
    影像方向
    選擇影像的方向。您可以選擇“關閉旋轉”或“旋轉90度”。預設值為“關閉旋轉”。
    雙面影印
    選擇單面或雙面原稿。
    可選擇單面，雙面(書本式)或雙面(便簽式)。預設值為“單面”。
    檔案格式
    可以設定掃描檔案格式（檔案類型，壓縮模式，和壓縮率）。可以設定以下項目。
    “彩色/灰階”/“黑白”
    可以選擇掃描格式（彩色/灰階，或黑白）。
    檔案類型
    可以選擇掃描檔案類型。
    當在上述設定選擇“彩色/灰階”，則有PDF，PDF/A，TIFF和JPEG檔案類型可供選擇。
    當在“色彩”中選擇“黑白”，則有PDF，PDF/A和TIFF檔案類型可供選擇。
    壓縮式PDF
    當在“彩色/灰階”/“黑白”設定中選擇“彩色/灰階”，並且在“檔案類型”中選擇“PDF”或“PDF/A”則可以選擇壓縮式PDF（無，壓縮，或超細緻壓縮）的類型。
    注意：壓縮式PDF只當掃描器安裝掃描擴充套件時可選擇。
    壓縮比例
    當在“彩色/灰階”/“黑白”中選擇“彩色/灰階”時，可以選擇壓縮率（低，中，高）。此外若掃描器支援黑字增強PDF和“檔案類型”設定為“PDF”則可以選擇“黑字增強”。
    壓縮模式
    當在“色彩”中選擇“黑白”時，可以選擇壓縮模式（無, MH(G3), MMR(G4)）。
    每檔所指定頁面
    此功能當您選擇的檔案類型為“PDF”，“PDF/A”，或“TIFF”時才可使用。當選擇使用時，掃描的文件會拆分指定的頁面數。（例如指定2頁，掃描文件會以2頁方式分組來建立多頁檔案。）
    加密
    當檔案類型選擇“PDF”，可使用加密PDF。當勾選時，必須指定開啟加密PDF的“密碼”。
    解析度
    掃描解析度可以設定。下列為可供選擇的解析度。
    100x100dpi, 150x150dpi, 200x200dpi, 300x300dpi, 400x400dpi, 600x600dpi
    注意：只能在支援的掃描器選擇150x150dpi 的解析度。
    注意：在“壓縮式PDF”中當選擇“壓縮”或“超細緻壓縮”時，只能使用300x300dpi。
    其他設定
    額外的掃描設定歸類於“其他”。可以選擇以下項目：
    曝光
    可選擇掃描的曝光（自動，文字，文字/印刷相片，印刷相片，文字/相片，照片，地圖）。
    曝光度
    若“曝光度”設定為“自動”則可選擇掃描曝光度（1到5）。
    空白頁跳過
    當設定不為“關閉”時，若原稿中包含空白頁或背景陰影則掃描器當掃描時會自動刪除或跳過。設定選項為“關閉”，“空白頁跳過”，和“跳過空白和濾除背景“。
  2. 當您完成所有設定後，輕按“掃描從（掃描器名稱）”（在iPad上輕按“掃描”）開始掃描文件。
  3. 在輕按“掃描”按鈕之後，在掃描器操作面板會顯示目標的掃描器資訊（“顯示名稱”和“驗證碼”）並畫面切換到等待階段（準備掃描）。
  4. 確認裝置資訊和在掃描器上設定文件。
  5. 在彈出式對話框中選擇“掃描”然後繼續。
  6. 掃描資料會傳送到本應用程式並且資料會儲存到記憶體中。
    若只接收單份文件，資料的內容會顯示出來。
    至於預覽的操作，請參閱 預覽檔案

    選擇欲列印的印表機並設定列印設定。

  2. 會顯示在本應用程式儲存的檔案和資料夾清單。
  3. 移動到您欲建立資料夾的子資料夾下，然後輕按"編輯"按鈕。
  4. 核選框會顯示於清單的左側且選單列顯示於螢幕底部。在選單列輕按。
  5. 會顯示"建立資料夾"對話框。輸入資料夾名稱。
    輕按"儲存"來建立資料夾。

從外部應用程式列印

從行動裝置中其他應用程式所傳送的資料可以從本應用程式列印出來。
  1. 在其他應用程式中開啟您想要列印的檔案(JPEG,TIFF,PNG或PDF)。
  2. 在本應用程式操作"傳送檔案到其他應用程式"。(詳細操作請參考本應用程式的操作手冊)。
    "SharpdeskM"將會是在清單中的其中一個應用程式可以接收此檔案。然後輕按"SharpdeskM"。
    注意：若您在 iOS 4.3 到 5使用本應用程式則最多僅列出9個應用程式。若“Sharpdesk的M”沒有在清單內，請解除安裝不必要的應用程式。
  3. 剩餘的操作與在應用程式中列印儲存文件步驟5及其之後步驟相同。
