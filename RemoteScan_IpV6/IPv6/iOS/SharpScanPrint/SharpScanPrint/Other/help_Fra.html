Réglage des périphériques

Réglez des imprimantes et des scanners pour imprimer et numériser depuis cette application

Rechercher périphériques

  1. Appuyez sur "Réglages" en haut du menu.
  2. Appuyez sur "Réglage des périphériques".
  3. Appuyer sur "Rechercher périphérique(s)". Les imprimantes et scanners présents sur le réseau sont automatiquement découverts.
    (Remarque) Les périphériques doivent être sur le même segment du réseau auquel votre appareil est connecté. Si vous utilisez des périphériques sur des segments différents du réseau, référez-vous à     Ajouter un périphérique manuellement.
  4. Les imprimantes et scanners trouvés sur le réseau sont affichés sous forme de liste.
    Les informations "Produit" et "Emplacement" obtenues du périphérique sont affichées. "Nom du produit (Emplacement)" ou "Nom du produit (Adresse IP)" sont affichés au-dessus de l'"Adresse IP".
    (Remarque) Le format d'affichage ("Nom du produit (Emplacement)" ou "Nom du produit (Adresse IP)") de la liste peut être configuré depuis Réglages d'application. "SHARP" contenu dans le "Produit" obtenu du périphérique n'est pas affiché.

Ajouter un périphérique manuellement

  1. Appuyez sur "Réglages" en haut du menu.
  2. Appuyez sur "Réglage des périphériques".
  3. Appuyez sur "Ajouter manuellement" si vous souhaitez ajouter des périphériques qui ne sont pas dans le même segment de réseau que votre appareil (non trouvés avec Rechercher périphériques).
  4. Les éléments suivants peuvent être définis sous la rubrique "Information Imprimante / Scanner" et "Paramètres de comportement».
    Appuyez sur "Sauvegarder" pour enregistrer les réglages. Si vous souhaitez annuler cet ajout, appuyez sur le bouton "Retour" avant d'appuyer sur "Sauvegarder".
    Nom
    Entrez le nom de l'imprimante / scanner que vous souhaitez ajouter. Si aucun nom n'est entré, l'adresse IP de l'appareil apparaîtra comme nom.
    Adresse IP
    Entrez l'adresse IP du périphérique, ce champ est obligatoire.
    (Remarque) Une adresse IP ayant déjà été ajoutée manuellement ne peut pas être entrée.
    Numéro de port
    Entrez le numéro de port du périphérique, ce champ est obligatoire.
    (Remarque) Un numéro de port est nécessaire pour imprimer depuis cette application. Entrez le même numéro de port que celui configuré sur l'imprimante. Si des numéros de ports différents sont entrés, cette application ne pourra pas imprimer.
    Pour vérifier le numéro de port sur l'imprimante,
    Si "Utiliser pour impression Raw" des Réglages d'application est décoché : Voir [Réglages d'application] - [Réglage pour l'impression à partir d'un PC] sur la page web de l'imprimante. Le port par défaut pour cette application et l'imprimante est le port 21.
    Si cette option est cochée, Voir l'onglet [Réglages réseau] - [Réglages de l'impression directe] - [RAW] sur la page Web de l'imprimante. Le numéro de port par défaut est 9100.
    Choisir ce périphérique par défaut
    Si "Choisir ce périphérique par défaut" est coché, ce périphérique sera choisi par défaut lorsque vous imprimerez ou numériserez.
    (Remarque) "Choisir ce périphérique par défaut" ne peut pas être configuré si "Sélection automatique du périphérique" est coché dans Réglages d'application.

Voir et modifier les informations du périphérique

  1. Appuyez sur "Réglages" en haut du menu.
  2. Appuyez sur "Réglage des périphériques".
  3. Les périphériques enregistrés sont affichés sous forme de liste.
    L'icône indique les périphériques ajoutés automatiquement.
    L'icône indique les périphériques ajoutés manuellement.
  4. Appuyez sur un périphérique pour visualiser ses informations.
  5. Les informations relatives au périphérique sélectionné sont affichées. Les éléments affichés ici sont disponibles dans ajouter des imprimantes / scanners manuellement .
    (Remarque) Lors d'un ajout automatique de périphérique, l'adresse IP n'est pas modifiable.
    Lors d'un ajout manuel de périphérique, le nom du produit et son emplacement ne sont pas modifiables. Le nom du produit et les champs de localisation sont automatiquement remplis lorsque l'application établit un premier contact avec l'imprimante / scanner.
  6. Lorsque "numériser depuis un périphérique mobile" est disponible, les paramètres supplémentaires suivants sont disponibles sous "Paramètres de fonctionnement".
    Numériser depuis votre appareil mobile
    Lorsqu'il est activé ("I (ON)"), configurez les paramètres de numérisation et numérisez sur votre appareil mobile.
    Nom affiché
    Le nom d'affichage s'affiche avec un code de vérification à la fois sur le panneau de commande du scanner et sur votre appareil mobile pour s'assurer du scanner cible.
    Générer automatiquement des codes de vérification
    Lorsqu'il est activé ("I (ON)"), génére automatiquement un code de vérification. Si c'est désactivé ("O (OFF)"), vous devez entrer un code dans le champ Code de vérification.
    Code de vérification
    Le code de vérification sera affiché avec un nom d'affichage à la fois sur le panneau de commande du scanner et votre appareil mobile pour s'assurer du scanner cible.
    Réinitialiser les réglages
    Tapez pour réinitialiser les paramètres de numérisation configurés lors de la dernière opération de numérisation.
  7. Appuyer sur "Enregistrer" pour enregistrer les réglages.

Déplacer ou Supprimer des périphériques de la liste.

  1. Appuyez sur "Réglages" en haut du menu.
  2. Appuyez sur "Réglage des périphériques".
  3. Les périphériques enregistrés sont affichés sous forme de liste.
  4. Appuyez sur "Editer".
  5. Appuyez sur puis sur "Supprimer" si vous souhaitez supprimer un périphérique de la liste.
  6. Appuyez sur puis déplacer votre doigt vers le haut ou le bas pour changer l'ordre des périphériques.
    (Remarque) Les périphériques supprimés peuvent être ajoutés à nouveau avec     Rechercher périphériques ou Ajouter un périphérique Manuellement.

Réglage info. util.

Réglez les informations de l'utilisateur.
  1. Appuyez sur "Réglages" en haut du menu.
  2. Appuyez sur "Réglage des informations utilisateur"
  3. "Informations du profile" est le réglage nécessaire pour l'enregistrement d'un destination sur un scanner pour l'opération de numérisation. Les réglages suivants sont disponibles :
    Nom affiché
    Affiché dans le carnet d'adresse en tant que destination. La valeur par défaut est la même que la valeur de "Nom" configurée dans les informations utilisateur de l'appareil.
    Initale
    "Initiale" est utilisé comme clé de tri sur l'affichage du "Nom Affiché" dans l'écran du carnet d'adresses du scanner.
  4. Authentification utilisateur ("Nom de connexion" et "Mot de passe") est le réglage nécessaire pour imprimer sur une imprimante où le contrôle de l'utilisateur est activé. Les réglages suivants sont disponibles :
    Utiliser le nom de connexion pour l'authentification
    Sélectionnez cette option lorsque le nom de connexion est utilisé pour l'authentification des utilisateurs sur les imprimantes. Tapez "Nom de connexion" et "Mot de passe".
    Nom de connexion
    Entrez votre nom d'utilisateur utilisé pour l'authentification des utilisateurs sur les imprimantes.
    Mot de passe
    Entrez votre nom d'utilisateur utilisé pour l'authentification des utilisateurs sur les imprimantes.
    Utiliser le numéro d'utilisateur pour l'authentification
    Sélectionnez cette option lorsque le numéro d'utilisateur est utilisé pour l'authentification des utilisateurs sur les imprimantes de votre réseau. Tapez dans "Numéro d'utilisateur".
    N° d'utilis.
    Entrez votre numéro d'utilisateur utilisé pour l'authentification des utilisateurs sur les imprimantes.
  5. Lorsque "ID du travail par défaut" est sélectionné, les imprimantes affichent "Nom d'utilisateur" et "Nom du travail" configuré ici sur le panneau de commande.
    Nom util.
    Saisissez un nom d'utilisateur qui sera affiché sur le panneau de commande de l'imprimante. Si ce n'est pas défini, l'imprimante utilisera la valeur "SharpdeskM utilisateur".
    Nom du travail
    Entrez un nom de travail qui sera affiché sur le panneau de commande de l'imprimante. Si ce n'est pas réglé, l'imprimante utilisera la valeur, "Travail SharpdeskMobile".
  6. Appuyer sur "Enregistrer" pour enregistrer les réglages.

Réglages d'application.

Paramétrer le fonctionnement de l'application.
  1. Appuyez sur "Réglages" en haut du menu.
  2. Appuyer sur "Réglages d'application".
  3. Les réglages suivants peuvent être réglés depuis "Réglages d'application".
    Effacement automatique du profil
    Sélectionnez "I(ON)" dans "Effacement automatique du profil" si vous souhaitez supprimer la destination sur le scanner une fois la numérisation terminée.
    Ecrasement du profil
    Sélectionnez sur "I(ON)" pour écraser la destination enregistrée lors de la dernière numérisation sans prévenir.
    Sauvegarde de fichier reçu d'une autre application
    Sélectionnez "I(ON)" Si vous voulez que Sharpdesk Mobile conserve le fichier reçu d'une autre application lors de l'impression de ce fichier.
    Sélection automatique du périphérique
    Sélectionnez "I(ON)" pour sélectionner le périphérique par défaut lié au réseau Wi-Fi (SSID).
    Impression haute qualité
    Sélectionnez "I(ON)" pour imprimer en haute qualité.
    Utiliser port Raw
    Sélectionnez "I(ON)" pour utiliser un port RAW (le numéro de port par défaut est 9100) à la place du port FTP (le numéro de port par défaut est 21) pour l'impression.
    Nom par défaut pour périphériques ajoutés automatiquement
    Le format d'affichage peut être réglé pour les imprimantes / scanners qui sont automatiquement ajoutés. "Nom du produit (Lieu)" ou "Nom du produit (adresse IP)" peuvent être sélectionnés.
  4. Il est nécessaire de configurer les "Réglages SNMP" si le réseau sur lequel est connecté votre périphérique utilise d'autres paramètres que les paramètres de la communauté par défaut SNMP GET. Régler sur "O(OFF)" dans "Chercher par Public" si vous utilisez cette application sur un tel réseau, et réglez les chaines qui sont utilisées dans ce réseau dans "Chaine(s) de communauté". 10 chaînes peuvent être entrées. Veuillez demander la chaîne de communauté SNMP GET à votre administrateur réseau.
  5. Appuyer sur "Enregistrer" pour enregistrer les réglages.

Réglages E-mail

Réglez votre compte de messagerie pour recevoir et imprimer des e-mails depuis cette application.
  1. Appuyez sur "Réglages" en haut du menu.
  2. Appuyez sur "Réglages e-mail"
  3. Les réglages suivants sont disponibles pour le compte de messagerie
    Nom du compte
    Entrez le nom d'utilisateur pour vous connecter à votre compte de messagerie
    Ce champ est obligatoire quand un mot de passe est défini
    Mot de passe
    Entrez le mot de passe pour vous connecter à votre compte de messagerie
  4. Les réglages suivants sont disponibles pour le serveur de messagerie. Contactez votre administrateur pour les informations du serveur.
    Remarque : Le serveur de messagerie doit supporter IMAP4
    Nom d'hôte
    Entrez le nom du serveur de messagerie
    Numéro de port
    Entrez le numéro de port du serveur de messagerie. Ce champ est obligatoire. Le numéro de port par défaut est 143 si "SSL" n'est pas coché, et 993 si il est coché.
    SSL
    Vérifiez si SSL est requis pour connecter votre appareil mobile au serveur de messagerie
  5. Appuyer sur "Enregistrer" pour enregistrer les réglages.

Scanner

Cette fonction reçoit et stocke les documents numérisés.
  1. Appuyez sur "Scanner" sur l'écran principal
  2. Appuyez sur "Scanner: (nom du scanner)" (sur iPad, appuyez sur le bouton en dessous de "Scanner") pour sélectionner un scanner.
    Le scanner dont l'icône est est le scanner par défaut.
Les opérations peuvent varier en fonction du scanner.
Veuillez vous rendre sur depuis votre appareil mobile pour apprendre à exécuter un scan depuis votre appareil mobile. Si l'option de réglage de numérisation n'est pas disponible sur votre appareil mobile, s'il vous plaît allez à Numérisation à partir du panneau de commande du scanner .

Numériser depuis votre appareil mobile

Veuillez suivre les étapes suivantes pour configurer les paramètres de numérisation sur votre appareil mobile. Aller à le site Web de l'application pour la liste des scanners supportés.
Toutes les opérations, sauf «la fixation d'un document sur le scanner" doivent être terminées dans l'application.
Note: Veuillez confirmer que l'écran d'accueil ou l'écran d'authentification utilisateur s'affiche sur le scanner. Lorsque l'authentification externe est activé, l'écran d'authentification utilisateur s'affiche. Une fois l'authentification réussie, l'écran d'accueil scanner s'affiche.
  1. Si vous avez sélectionné le scanner qui prend en charge la numérisation à partir des appareils mobiles, les menus de configuration et les paramètres de numérisation actuels seront affichés.
    Remarque: Lorsque vous désactivez cette option, vous pouvez lancer la numérisation depuis le scanner. S'il vous plaît allez à numérisation depuis le panneau de commande du scanner pour plus d'instructions.
    Les paramètres suivants sont disponibles. Si vous utilisez l'iPhone, les réglages peuvent être mémorisés en appuyant sur "Terminer". Si vous utilisez l'iPad, les paramètres sont automatiquement enregistrés.
    Réglages Mode Couleur
    Réglages pour numériser les originaux en couleur (Auto/Couleur/Niveaux de gris/Noir et blanc). La valeur par défaut est "Auto".
    Remarque : Quand "Grande taille" est sélectionné pour "Format de l'original", sous "Original", seul "Mono2" est disponible.
    Original
    Réglages du document original à numériser. Les réglages suivants peuvent être faits :
    Format de l'original
    Lorsque "Auto" est réglé par défaut, la taille du papier d'origine non-standard ne peut pas être détectée par le scanner. Vous devez changer manuellement le réglage de la taille originale.
    Format papaier détecté (affichage uniquement)
    Lorsque vous définissez un document sur le scanner cible, le format de papier original sera automatiquement détecté et affiché sur votre appareil mobile et sur ​​le panneau de commande du scanner.
    Taille enreg.
    La taille de l'image numérisée peut être réglée manuellement. Si cette taille est différente de la taille de papier détecté, l'image est automatiquement ajustée pour s'adapter à la page. La valeur par défaut est "Auto".
    Note: Vous ne pouvez sélectionner "Auto" lorsque l'un des formats de papier originaux suivants est sélectionné:Grande Taille / Carte postale japonaise / Carte d'affaires/2L Taille / Carte / Personnalisé.
    Enregistrement du format personnalisé
    Des formats de papier originaux personnalisés peuvent être enregistrés. Jusqu'à 5 tailles personnalisées peuvent être enregistrées.
    Orientation
    Sélectionner l'orientation de l'image. Vous pouvez sélectionner «Pas de rotation" ou "rotation de 90 degrés". La valeur par défaut est "Pas de rotation".
    Copie recto verso
    Sélectionnez recto ou recto-verso pour l'original.
    recto, recto-verso(livre) ou recto-verso(bloc-note) peuvent être sélectionnés. La valeur par défaut est "recto",
    Format fichier
    Le format du fichier de numérisation (type de fichier, le mode de compression, et le taux de compression) peut être réglé. Les éléments suivants peuvent être réglés.
    "Couleur/Niveaux de gris"/"Noir et blanc"
    Mode de numérisation (Couleur/Niveaux de gris, ou noir et blanc) peuvent être sélectionnés.
    Type de fichier
    Le type de fichier de numérisation peut être sélectionné.
    Lorsque "Couleur / Niveaux de gris" est sélectionné dans le réglage ci-dessus, les types de fichiers JPEG, PDF, PDF/A, TIFF ou JPEG sont disponibles.
    Si "B / W (monochrome)" est sélectionné dans "Couleur", les types de fichiers PDF, PDF / A, et TIFF sont disponibles.
    PDF Compact
    Le type de PDF Compact (Aucun, Compact ou Compact Ultra Fin) peut être sélectionné quand "Couleur / Niveaux de gris" est sélectionné dans le réglage "Couleur / Niveaux de gris" /"Noir et blanc" et quand "PDF" ou "PDF/A" est sélectionné dans "Type de fichier".
    Remarque : Compact PDF est disponible uniquement si le kit d'extension scanner est installé dans le scanner.
    Taux compres.
    Le Taux de compression (faible, moyenne, élevée) peut être sélectionné quand "Couleur / Niveaux de gris" est sélectionné dans le réglage "Couleur / Niveaux de gris" / "Noir et blanc". En outre, "accentuer les caractères noirs" peut être sélectionné si "PDF" est défini comme le "Type de fichier".
    Mode de compres.
    Le mode de compression (aucune, MH (G3), MMR (G4)) peut être sélectionné quand "Noir et blanc" est sélectionné dans "Couleur".
    Pages spécifiées par fichier
    Cette fonction est disponible lorsque vous sélectionnez "PDF", "PDF / A" ou "TIFF" comme type de fichier. Une fois sélectionné, le document numérisé est divisée en fichiers par nombre spécifié de pages. (Par exemple, lorsque 2 pages est spécifié, le document numérisé est regroupé par 2 pages pour créer plusieurs fichiers.)
    Cryptage
    Quand type de fichier "PDF" est sélectionné, l'option PDF crypté est disponible. Lorsqu'elle est cochée, un "Mot de passe" doit être assigné pour ouvrir le PDF chiffré.
    Résolution
    La résolution de numérisation peut être réglé. Les options de résolution ci-dessous sont disponibles.
    100x100ppp, 150x150ppp, 200x200ppp, 300x300ppp, 400x400ppp, 600x600ppp
    Remarque : la résolution 150x150 ppp ne peut être sélectionnée que sur les scanners qui prennent en charge cette résolution.
    Remarque : Seule la résolution 300x300 ppp est disponible lorsque vous sélectionnez "Compact" ou "Compact Ultra Fin" sous "PDF compact".
    Autres
    Des paramètres de numérisation supplémentaires sont disponibles sous la rubrique "Autres". Les éléments suivants sont disponibles:
    Exposition
    L'exposition (Auto, Caractères, Texte / Photo imp, photo imp, texte / photo, photo, carte) peut être sélectionnée.
    Niveau d'exposition
    Le niveau d'exposition (1 à 5) peut être sélectionné. Ceci peut être réglé si "Auto" est sélectionné pour "exposition".
    Suppr. pages vierges
    Si réglé sur une valeur autre que "Off" et que l'original contient des pages blanches ou des ombres en arrière-plan, le scanner les détecte automatiquement et -les ignore lors de la numérisation. Les options de réglage sont "Sans", "Ignorer les pages vierges» et «Ignorer les pages vierges et Ombres".
  2. Lorsque vous avez terminé tous les réglages, appuyez sur "Numériser depuis (nom du scanner)" (sur l'iPad, appuyez sur "Numériser") pour lancer la numérisation du document.
  3. Après avoir appuyé sur le bouton "Numériser", l'information du scanner cible ("Display Name" et "Code de vérification") sera affiché sur le panneau de commande du scanner et passera à une phase d'attente.
  4. Confirmer les informations du périphérique et placer le document sur le scanner.
  5. Sélectionnez "Numériser" dans la fenêtre pour continuer.
  6. Les données numérisées sont envoyées à cette application, qui a stocké les données dans la mémoire.
    Si une seule donnée est reçue, son contenu est affiché.
    Pour le fonctionnement de la prévisualisation, aller à aperçu d'un fichier
  7. Appuyez sur le bouton "Enregistrer cette image".
  8. Les données numérisées, déjà nommés au scanner, peuvent être renommés à ce stade
    Si vous le souhaitez, saisissez un nouveau nom de fichier et appuyez sur "Enregistrer".
  9. Sélectionnez le dossier de destination et appuyez sur "Sauvegarder ici" en bas de l'affichage pour enregistrer les données en tant que fichier dans le dossier.
Si plusieurs données numérisées sont reçues, suivez les instructions suivantes:
  • Les données reçus sont listées. Appuyez sur "Sauvegarde" dans le menu, et suivez les étapes 7 à 8. pour sauvegarder ces données en tant que fichiers dans le dossier sélectionné du périphérique mobile. (Un suffixe séquentiel est ajouté au nom du fichier entré à l'étape 7.)
    (Remarque) Appuyez sur les données de la liste afin de les prévisualiser. Cependant, ces données ne peuvent être stockées qu'avec le reste des données reçues.

Numériser depuis le panneau du Scanner

Si "Scan de votre appareil mobile" n'est pas disponible sur le scanner, utilisez les étapes suivantes pour numériser à partir du panneau de commande du scanner.
Le format de fichier, la résolution et les autres paramètres de numérisation doivent être faits depuis le panneau de commande du MFP lors de la numérisation. (voir étape 3)
(Remarque) Assurez-vous que le carnet d'adresse ne s'affiche pas sur le panneau de commande du scanner. Si c'est le cas, cette application peut ne pas être en état d'attente. (Voir l'étape suivante)
  1. Appuyez sur "Scanner à partir de (nom du scanner)" (sur iPad, appuyez sur "Scanner").
  2. Appuyez sur "OK" lorsque le message "Voulez-vous scanner?" s'affiche. Puis un autre message s'affiche avec le "nom affiché" et l"initiale". Cette application est ensuite en état d'attente.
  3. Placez le document sur ​​le scanner puis configurez le format de fichier, la résolution, le mode couleur ou d'autres paramètres.
    Sélectionnez la destination du "Carnet d'adresses" dans "Envoi d'image" afin que la destination soit la même que le "Nom affiché" affiché dans le message du périphérique de l'étape précédente.
  4. Appuyez sur le bouton départ du scanner pour commencer la numérisation. Commencez à l'étape 6 numérisation depuis votre appareil mobile (Remote Scan) pour terminer le processus.

Imprimer

Les documents stockés dans l'application, les photos, pages Web et e-mails reçus peuvent être imprimés.

Imprimer un document enregistré dans l'application

Les formats supportés sont JPEG, PDF, PNG (ainsi que les PDF compressés et les PDF cryptés) et TIFF
  1. Appuyez sur "Imprimer" depuis l'écran principal
  2. Sur iPhone, appuyez sur "Sélectionner un fichier"
  3. Les fichiers et dossiers enregistrés sont listés. Voir modifier la liste des fichiers / dossiers pour modifier la liste des fichiers / dossiers ou rechercher des fichiers / dossiers pour rechercher des fichiers / dossiers.
  4. Sélectionnez un fichier pour afficher un aperçu avec les paramètres d'impression.
    Pour le fonctionnement de la prévisualisation, aller à aperçu d'un fichier
  5. Les réglages suivants sont disponibles :
    Imprimante
    Sélectionnez l'imprimante sur laquelle imprimer, et configurez les paramètres d'impression.
    L'imprimante dont l'icône est est l'imprimante par défaut.
    Quantité
    Si nécessaire, appuyez sur "Quantité" pour configurer la quantité d'impression. Vous pouvez configurer la quantité de 1 à 99 jeux.
    Réglages Mode Couleur
    Appuyez sur "Mode Couleur" pour choisir le mode couleur. Les modes Auto, Couleur, ou Niveaux de Gris peuvent être sélectionnés.
    Réglages Format Papier
    Appuyez sur "Format Papier" pour configurer le format papier de l'impression. Les formats suivants peuvent être sélectionnés:
    A3W, A3, A4, A5, B4, B5, 11 x 17, Letter, 14 x 8.5, 10.5 x 7.25, 8.5 x 11, 13 x 8, 8K, 16K, DL, C5, COM10, 10.5 x 7.25, Carte Postale Japonaise, 240 x 332, Choukei 3, Japanese You #2, Japanese You #4
    Réglage recto-verso
    Glissez le menu du bas et appuyez sur "Réglage recto-verso" pour configurer l'impression recto uniquement ou recto-verso. En ce qui concerne l'impression recto-verso, elle peut être configuré conjointement au sens de la reliure : bloc-notes ou livre.
    Réglages N-Up
    Tapez pour réduire la taille de l'image et faire tenir plusieurs pages sur une seule page.
    N-Up: 1-Up (1 page par une seule feuille), 2-Up (2 pages par une simple feuille) ou 4 pages (4 pages par une simple feuille) peuvent être sélectionnés.
    Ordre : Pour l'impression 2-Up sélectionner "gauche à droite" ou "droite à gauche". Pour l'impression 4-Up sélectionner "droite et bas", "vers le bas, et à droite", "gauche et vers le bas» ou «vers le bas, et à gauche".
    Remarque : Vous ne pouvez pas spécifier le réglage N-Up pour les fichiers TIFF, PDF compact, Bk letter et PDF autres que ceux numérisés sur un scanner Sharp.

    Configurer la plage à imprimer.
    Toutes les pages : Imprimer toutes les pages

    Entrée directe: Entrez les numéros de page et / ou les plages de pages. Pour les plages de pages, utilisez un trait d'union. Vous pouvez séparer chaque page ou plage avec une virgule.
    Remarque : Vous ne pouvez pas spécifier une zone d'impression pour les fichiers PDF compact, Bk letter et PDF autres que ceux numérisés sur un scanner Sharp.
    Rétention
    Appuyez sur "rétention" pour maintenir une tâche d'impression sur le disque dur de l'imprimante, ce qui permet d'imprimer le travail à partir du panneau de commande de l'imprimante en cas de besoin. Réglez "Maintien uniquement" à "ON" pour activer cette fonction. Le dossier principal est défini comme l'emplacement du fichier maintenu.
    Si "Spécifier Mot de passe" est réglé sur "ON", le mot de passe (de 5 à 8 chiffres) permet de stocker un travail confidentiel sur le dispositif d'impression et le fichier ne peut être imprimé à moins que le mot de passe correct ne soit entré sur le panneau du MFP.
    Remarque : Vous ne pouvez pas spécifier de rétention pour un fichier PDF crypté. Il peut être imprimé de manière sécurisée en exécutant les opérations à partir de l'étape 7.

    (Remarque) Si l'imprimante sélectionnée ne dispose pas des options appropriées pour imprimer le fichier, une erreur s'affichera. Veuillez sélectionner une autre imprimante.
Si vous imprimez un fichier PDF crypté, suivez les étapes suivantes :
(Remarque) Les opérations suivantes suppose l'utilisation d'un MX-2610N/3110N/3610N.

  4. Le travail d'impression envoyé depuis l'application est affiché. Sélectionnez le travail, et appuyez sur "Entrer le mot de passe pour le fichier PDF crypté" sur le tableau de bord.

Organisation

Les fichiers ou dossiers stockés dans l'application peuvent être gérés. Les actions disponibles incluent : lister, rechercher, prévisualiser, modifier le nom, déplacer, supprimer ou créer.

Modifier la liste des fichiers et dossiers

Vous pouvez rechercher des fichiers / dossiers ou trier les fichiers / liste des dossiers.
  1. Appuyez sur le bouton "Organiser" depuis l'écran principal
  2. Les fichiers et les dossiers sont listés.
  3. Le bouton Trier et la barre de recherche sont affichés dans la liste des fichiers / dossiers (pour rechercher des fichiers / dossiers, voir rechercher des fichiers / dossiers ).
    Une des icônes suivantes exprimant la condition de tri et l'ordre actuel est indiquée dans le bouton Trier.
    4. Icône Condition de tri et ordre
    Ordre croissant d'horodatage
    Ordre décroissant d'horodatage
    Ordre croissant de nom de fichier
    Ordre décroissant de nom de fichier
    Ordre croissant de taille de fichier
    Ordre décroissant de taille de fichier
    Ordre croissant de format de fichier
    Ordre décroissant de taille de fichier
  4. Appuyez sur le bouton de tri pour trier. La condition de tri et l'ordre peuvent être changés. Les conditions de tri comprennent l'horodatage, le nom, la taille et le format de fichier. Tapez pour l'ordre croissant et pour l'ordre décroissant.

Rechercher des fichiers / dossiers

  1. Appuyez sur le bouton "Organiser" depuis l'écran principal
  2. Les fichiers et dossiers enregistrés sont listés.
  3. Le bouton Trier et la barre de recherche sont affichés dans la liste des fichiers / dossiers (pour trier des fichiers / dossiers, voir "Modifier la liste des fichiers / dossiers").
  4. Entrez le nom de fichier complet ou une partie du nom de fichier que vous souhaitez trouver dans la barre de recherche. Une liste de fichiers contenant les caractères recherchés s'affiche.
  5. La poursuite des recherches à partir du résultat peut être effectuée en tapant sur "Recherche avancée" à droite de la barre de recherche.
    Les conditions suivantes peuvent être spécifiées pour affiner le résultat de la recherche.
    Inclure les sous-dossiers
    Inclure les sous-dossiers
    Dossier
    Inclure les dossiers
    PDF
    Inclure les PDF
    TIFF
    Inclure les TIFF
    Fichier image (JPEG,PNG)
    Inclure les PNG ET JPEG
  6. Vous pouvez modifier l'ordre d'affichage des résultats de recherche. Voir Modifier la liste des fichiers / dossiers.

Prévisualiser un fichier

  1. Appuyez sur le bouton "Organiser" depuis l'écran principal
  2. Les fichiers et dossiers enregistrés sont listés.
  3. Appuyez sur un fichier pour le prévisualiser
    (Remarque) La prévisualisation n'est pas possible si la mémoire du périphérique mobile n'est pas suffisante.
    Les PDF cryptés ne peuvent pas être prévisualisés. Si vous souhaitez les prévisualiser, appuyez sur "Vérifier avec une autre application" pour sélectionner l'application appropriée.
  4. Utiliser deux doigts pour zoomer / dézoomer l'image prévisualisée.
    Appuyez sur pour faire pivoter la prévisualisation de 90 degrés dans le sens horaire.
    (Remarque) Les fichiers avec le format suivant peuvent être pivotés :
    • JPEG, TIFF, PNG, PDF numérisé avec un scanner SHARP (sauf PDF crypté).
    (Remarque) L'orientation et/ou la taille originale est conservée si une autre opération est effectuée après avoir fait pivoter la prévisualisation.

Renommer un fichier/dossier

  1. Appuyez sur le bouton "Organiser" depuis l'écran principal
  2. Les fichiers et dossiers enregistrés sont listés. Appuyez sur le bouton "Editer".
  3. Des cases à cocher apparaissent à gauche de la liste, et la barre de menu apparait en bas.
  4. Cochez la case du fichier/dossier que vous souhaitez renommer, puis appuyez sur dans la barre de menu.
    Remarque : Vous ne pouvez renommer qu'un fichier / dossier à la fois.
  5. Le message "Renommer Fichier(Dossier)" apparait. Entrez un nouveau nom.
    Appuyez sur "Enregistrer" pour renommer.
    (Remarque) L'extension du fichier ne peut pas être changée.

Déplacer des fichiers/dossiers

  1. Appuyez sur le bouton "Organiser" depuis l'écran principal
  2. Les fichiers et dossiers enregistrés sont listés. Appuyez sur le bouton "Editer".
  3. Des cases à cocher apparaissent à gauche de la liste, et la barre de menu apparait en bas.
  4. Cochez la case du fichier/dossier que vous souhaitez déplacer (plusieurs fichiers/dossiers peuvent être choisis) et appuyez sur dans la barre de menu..
  5. La fenêtre "Déplacer" apparait. Sélectionnez le dossier de destination et appuyer sur "Déplacer Ici"

Supprimer des fichiers/dossiers

  1. Appuyez sur le bouton "Organiser" depuis l'écran principal
  2. Les fichiers et dossiers enregistrés sont listés. Appuyez sur le bouton "Editer".
  3. Des cases à cocher apparaissent à gauche de la liste, et la barre de menu apparait en bas.
  4. Cochez la case du fichier/dossier que vous souhaitez supprimer (plusieurs fichiers/dossiers peuvent être choisis) et appuyez sur dans la barre de menu.
  5. La fenêtre "Supprimer fichier(s)/dossier(s)" apparait. Appuyer sur "OK" pour supprimer les fichiers/dossiers.
Vous pouvez aussi supprimer un fichier de la façon suivante :
  • A l'étape 2, appuyez sur le fichier choisi et faites-le glisser hors de l'écran puis appuyez sur "Supprimer".
  • Appuyez sur "Supprimer le ficher" en bas de l'écran de prévisualisation
    Remarque : Les dossiers ne peuvent pas être supprimés par cette opération.

Créer un dossier

  1. Appuyez sur le bouton "Organiser" depuis l'écran principal
  2. Les fichiers et dossiers enregistrés sont listés.
  3. Allez dans le dossier dans lequel vous souhaitez créer un dossier, puis appuyez sur le bouton "Editer".
  4. Des cases à cocher apparaissent à gauche de la liste, et la barre de menu apparait en bas. Appuyez sur dans la barre de menu.
  5. La fenêtre "Créer un dossier" est affichée. Entrez un nom de dossier.
    Appuyez sur "Enregistrer" pour créer le dossier.
