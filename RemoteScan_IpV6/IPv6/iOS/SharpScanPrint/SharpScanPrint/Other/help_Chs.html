设置打印机/扫描仪

设置从该应用程序打印的打印机以及该应用程序接收的扫描仪。

查找打印机/扫描仪

  1. 轻击主界面上的"设置"。
  2. 轻击"设置打印机/扫描仪。"
  3. 轻击"查找打印机/扫描仪。"网络上的打印机和扫描仪将被自动搜索出来。
    注意：打印机/扫描仪必须要在和您设备连接的网络相同的段中。如果您使用的打印机/扫描仪在不同的网段中，那么请参照    手动添加打印机/扫描仪。
  4. 找到的打印机和扫描仪显示在列表中。
    从打印机/扫描仪上获取的"产品名称"和"位置"显示。（“产品名称（位置）"或"产品名称（IP地址）"显示在IP地址上面）。
    注意：列表中的显示格式（"产品名称（位置）"或"产品名称（IP地址）"）可以在应用程序设置中配置。从打印机/扫描仪获取的"产品名称"中包含的"SHARP"不显示出来。

手动添加打印机/扫描仪

  1. 轻击主界面上的"设置"。
  2. 轻击"设置打印机/扫描仪。"
  3. 如果要添加的打印机/扫描仪不在您设备所在的网段（不能通过查找打印机/扫描仪添加），那么请轻击"手动添加"。
  4. 以下选项可以在“打印机/扫描仪信息”和“动作设置”中设置。
    轻击"保存"保存设置。如果要取消添加，轻击"保存"前按下屏幕外的“返回”键。
    名称
    输入您要添加的打印机/扫描仪的名称。若名称未输入，则设备IP地址将显示为名称。
    IP地址
    输入打印机/扫描仪IP地址。这是必填框。
    注意：已经手动添加过的IP地址不能输入。
    端口号
    为打印机/扫描仪输入端口号。这是必填框。
    从应用程序打印端口号是必须的。请输入与打印机配置相同的端口号。如果输入了不同的端口号，那么应用程序就不能打印到打印机。
    查看打印机上的端口号。
    如果应用程序设置中的"使用Raw打印"未选择：查看打印机网页中【应用程序设置】-【从PC进行打印的设置】，打印机和本程序的默认端口都是21。
    如果选择了：查看打印机网页中【网络设置】-【打印端口设置】-【RAW】标签。默认端口为9100。
    设置为默认打印机/扫描仪
    如果选择了"I（开）"，那么当你使用"打印"或"扫描"时，默认的打印机/扫描仪为这台打印机/扫描仪。
    注意：如果应用程序设置中"自动选择打印机/扫描仪"被选中，那么"设置为默认打印机/扫描仪"不能设置。

查看和编辑打印机/扫描仪信息

  1. 轻击主界面上的"设置"。
  2. 轻击"设置打印机/扫描仪。"
  3. 注册过的打印机/扫描仪显示在列表中。
    图标表示自动添加的打印机/扫描仪。
    图标表示手动添加的打印机/扫描仪。
  4. 轻击打印机/扫描仪查看信息。
  5. 选中的打印机/扫描仪信息显示出来。此处显示的项目在手动添加打印机/扫描仪中有效。
    注意：对于自动添加的打印机/扫描仪，IP地址是不能被编辑。
    对于手动添加的打印机/扫描仪，产品名称和位置是不能被编辑的。当程序和打印机/扫描仪建立首次连接时，产品名称和位置框就被自动填写了。
  6. “扫描自移动设备”可以时，“操作设置”下的以下额外设置可用。
    从您的移动设备扫描
    启用（选择“I（开）”）时，请配置扫描设置并在您的移动设备上扫描。
    显示名称
    显示名称会连同校验码显示在扫描仪操作面板和您的移动设备上来确认目标扫描仪。
    自动生成校验码
    启用（选择“I（开）”）时，会自动生成校验码。如果未启用（选择“O（关）”），那么您需要在校验码框中输入校验码。
    校验码
    校验码会连同显示名称显示在扫描仪操作面板和您的移动设备上来确认目标扫描仪。
    重置扫描参数
    轻击此处重置最后一次扫描操作配置的扫描设置
  7. 轻击"保存"来保存设置。

移动或删除列表中的打印机/扫描仪

  1. 轻击主界面上的"设置"。
  2. 轻击"设置打印机/扫描仪。"
  3. 注册过的打印机/扫描仪显示在列表中。
  4. 轻击"编辑"菜单。
  5. 如果您要从列表中删除打印机/扫描仪请轻击 然后再轻击"删除" 。
  6. 如果要给打印机/扫描仪排序，您可以轻击 并且上下移动手指。
    注意：删除的打印机/扫描仪可以通过    查找打印机/扫描仪或手动添加打印机/扫描仪再次添加。

设置用户信息

设置用户信息。
  1. 轻击主界面上的"设置"。
  2. 轻击"设置用户信息"。
  3. "配置信息"是注册目标地址到正在"扫描"操作的扫描仪时需要的设置。以下设置可用：
    显示名称
    作为扫描目标地址显示在扫描仪地址簿中。默认值和设备配置的用户信息上"名称"的值相同。
    首字母
    "首字母"被用于扫描仪地址簿屏幕中显示"显示名称"的分类键上。
  4. 打印到启用了用户认证的打印机上需要设置"用户认证"。以下设置可用：
    使用登录名认证
    打印机上的用户认证使用用户名时，请选择此处。请输入“登录名”和“密码”。
    登录名
    输入打印机上用户认证使用的登录名。
    密码
    输入打印机上用户认证使用的密码。
    使用用户编号认证
    网络中的打印机上的用户认证使用用户编号时，请选择此处。请输入“用户编号”。
    用户编号
    输入打印机上用户认证使用的用户编号。
  5. 选择“默认任务ID”时，打印机在操作面板上显示此处配置的“用户名”和“任务名”。
    用户名
    输入要在打印机操作面板上显示的用户名。若未设置，打印机使用“SharpdeskM User”。
    任务名称
    输入要在打印机操作面板上显示的任务名。若未设置，打印机使用要打印文件的文件名。
  6. 轻击"保存"来保存设置。

应用程序设置

设置应用程序操作。
  1. 轻击主界面上的"设置"。
  2. 轻击"应用程序设置"。
  3. 下面的设置可以在"操作设置"中设置。
    自动删除配置
    选择“I（开）”实现扫描完成后删除扫描仪上目标地址。
    自动更新配置
    选择“|（开）”实现覆盖最后一次扫描时注册的没有确认的目标地址。
    保存导入的文件
    选择“I（开）”实现保留从其它程序接收的要从本程序打印的文件。
    自动选择打印机/扫描仪
    选择“I（开）”实现自动选择默认的连接在设备所连接的Wi-Fi网络（SSID）中的打印机/扫描仪。
    高品质打印
    选择“I（开）”实现优质的打印效果。
    使用Raw打印
    选择“I（开）”实现使用Raw端口（默认端口号为9100）代替FTP端口（默认端口号为21）来打印。
    自动添加打印机/扫描仪的默认名称
    自动添加的打印机/扫描仪的显示格式可以设置。"产品名称（位置）"或者"产品名称（IP地址）"可被选择。
  4. 如果装置所连接的网络不同于默认的SNMP GET Community 字符串，那么需要设置"SNMP设置"。如果您使用的应用程序在这样的网络中，请设置"Public Get Community"为"O（关）"，然后设置网络中用到的"输入Community字符串"中的字符串。多达10个字符串可以被设置。关于SNMP GET Community符串请询问你的网络管理员。
  5. 轻击"保存"来保存设置。

邮件设置

设置邮件账号获取并从应用程序打印。
  1. 轻击主界面上的"设置"。
  2. 轻击“邮件设置”。
  3. 邮件账号的以下设置可用：
    帐号名称
    输入用户名登录邮件账户。
    设置密码后此框必填。
    密码
    输入密码登录邮件账户。
  4. 邮件服务器的以下设置可用。关于服务器信息请联系邮件管理员。
    注意：邮件服务器必须支持IMAP4。
    主机名称
    输入邮件服务器主机名。
    端口号
    输入邮件服务器端口号。此处必填。默认端口号为143（“SSL”未选择）或993（“SSL”选择）。
    SSL
    若连接您的移动设备到邮件服务器需要SSL，请选中。
  5. 轻击"保存"来保存设置。

扫描

该功能接收并保存扫描的文档。
  1. 轻击主屏幕上的"扫描"。
  2. 轻击"扫描仪：（扫描仪名称）"（iPad上轻击"扫描仪"下面的按钮）选择要用的扫描设备。
    图标 表示扫描仪为默认扫描仪。
不同扫描仪操作不同。
请转到 从移动设备扫描学习如何执行从移动设备扫描。若您的移动设备上的扫描设置选项不可用，请转到从扫描仪操作面板扫描。

从移动设备扫描

请完成下列步骤实现移动设备上的扫描设置。转到 应用程序网址获取支持的扫描仪列表。
除了“在扫描仪上放置文档”以外的所有操作都可以在应用程序上完成。
注意：请确认扫描仪上显示为扫描仪主界面或用户认证界面。当外部认证启用时，显示用户认证界面。成功认证后，显示扫描仪主界面。
  1. 若您选择了支持从移动设备扫描功能的扫描仪，则显示设置菜单和当前扫描设置。
    注意：若是关闭了此功能，您可以从扫描仪开始扫描。请转到从扫描仪操作面板扫描获得更多帮助。
    以下设置有效。若您使用iPhone,轻击“完成”保存设置。若您使用iPad，设置自动保存。
    色彩模式
    扫描彩色原稿的设置（自动/真彩/灰度/Mono2）。默认值为“自动”。
    注意：“原稿”下的“原稿尺寸”选为“长尺寸”时，仅“Mono2”可用。
    原稿
    关于扫描用的原稿文档的设置。以下项目可用：
    原稿尺寸
    当默认设置为“自动”时，非标准原稿纸张尺寸不能被扫描仪检知。您需要手动更改原稿尺寸设置。
    检知纸张尺寸（仅显示）
    您在目标扫描仪上设置文档时，原稿纸张尺寸能自动检知并显示在您的移动设备和扫描仪操作面板上。
    保存大小
    扫描的图像尺寸可以手动设置。若该尺寸不同于检知到的纸张尺寸，则图像自动调整到适应纸张。默认值为“自动”。
    注意：以下原稿尺寸选择时，您仅能选择“自动”：长尺寸/日本明信片/名片/2L尺寸/卡片/自定义尺寸。
    登录自定义尺寸
    自定义原稿纸张尺寸可登录。至多5个自定义尺寸可以登录。
    图像方向
    选择图像方向。您可以选择“关闭旋转”或“选择90°”。默认值为“关闭旋转”。
    双面复印
    选择单面或双面原稿。
    可选择单面、双面（书签）或双面（便签）。默认值为“单面”。
    文件格式
    扫描文件格式（文件类型、压缩模式及压缩率）可以设置。以下项目可设置。
    “真彩/灰度”/“黑白双色”
    可选择扫描模式（真彩/灰度，或黑白双色）。
    文件类型
    可选择扫描文件类型。
    在上面的设置中选择“真彩/灰度”时，PDF、PDF/A、TIFF和JPEG文件类型可用。
    “色彩”选择为“黑白双色（黑白的）”时，文件类型PDF，PDF/A，TIFF及JPEG可用。
    高压缩PDF
    “真彩/灰度”/“黑白双色”设置中选择了“真彩/灰度”时，压缩的PDF类型（无，压缩或高压缩）可选择。文件类型中选择为“PDF”或“PDF/A”。
    注意：压缩PDF仅在扫描仪上安装了扫描仪扩充组件后可用。
    压缩率
    “彩色/灰度”/“黑白双色”设置中选择了“彩色/灰度”时，压缩率（低，中，高）可以被选择。另外，若扫描仪支持黑字增强的PDF文件并且“文件类型”设置为“PDF”，则“黑字增强”可以选择。
    压缩模式
    “色彩”选择“黑白双色”时可选择压缩模式（无、MH(G3),MMR(G4))。
    指定每个文件的页数
    选择文件类型为“PDF”、“PDF/A”或“TIFF”时，此功能可用。选中后，扫描的文档被分割成多个指定页数的文件。（例如：指定2页后，扫描的文档分成2页一组创建多个文件。）
    加密
    选择文件类型为“PDF”时，加密的PDF选项可用。选中后，需要输入“密码”才能打开加密的PDF。
    分辨率
    可以设置扫描分辨率。以下分辨率可用。
    100x100dpi, 150x150dpi, 200x200dpi, 300x300dpi, 400x400dpi, 600x600dpi
    注意：150x150dpi仅能在支持此分辨率的扫描仪上选择。
    注意：“压缩PDF”选择“压缩”或“高压缩”时，仅300x300可用。
    其它设置
    其他扫描设置在“其他”中可见。以下项目可用：
    浓度
    可以选择扫描浓度（自动、文本、文本/打印照片、打印照片、文本/照片、照片、地图）。
    浓度水平
    可以选择扫描浓度水平（1-5）。“浓度”选择了“自动”后可设置。
    空白页跳过
    设置为非“关”时，若原稿包含空白页或投影到背面的原稿，则扫描时扫描仪自动检知并跳过。设置选项为“关”、“跳过空白页”、“跳过空白页及投影到背面的原稿”。
  2. 您完成所有设置后，请轻击“扫描自（扫描仪名称）”（iPad上轻击“扫描”）启动扫描文档。
  3. 轻击“扫描”按钮后，目标扫描仪信息（“显示名称”和“校验码”）显示在扫描仪操作面板上然后进入等待（扫描就绪）阶段。
  4. 确认设备信息然后在扫描仪上放置文档。
  5. 选择弹出框上的“扫描”继续。
  6. 扫描数据被发送到程序并保存在内存中。
    如果是单一数据被接收，那么数据预览就会显示出来。
    关于预览操作，转到预览文件。
  7. 轻击"保存图像"。
  8. 扫描的数据已经在扫描仪上命名，在此阶段可以重命名。
    根据需要，请输入新的文件名称然后轻击“保存”。
  9. 选择目标文件夹，轻击界面上的"保存到这里"按钮把这个作为文件的数据保存到文件夹中。
如果多个扫描数据被接收，操作下面的步骤：
  • 接收的数据以列表显示。轻击菜单中的"一揽子保存"，然后完成步骤7和8实现保存文件到选中的设备文件夹中。（一个序列号添加在步骤7输入的文件名称的末尾。）
    注意：轻击列表中任意数据可以预览。但是，该数据只能是和其它接收到的数据存储在一起的。

从扫描仪操作面板扫描

若“从移动设备扫描”在扫描仪上不可用，请使用一下步骤实现从扫描仪操作目标扫描。
扫描时分辨率，文件格式和其它扫描设置必须在MFP操作面板上完成。（参见步骤3）
注意：请确认地址簿没有显示在扫描仪操作面板上。若地址簿显示在操作面板，则此操作无效。
  1. 轻击"扫描自（扫描仪名称）"（iPad上轻击"扫描"）。
  2. 当出现对话框"执行扫描吗？"时，请轻击"好"。然后带有“显示名称”和“首字母”的对话框出现，表明程序进入等待状态。
  3. 在扫描仪上放置文档并选择“FTP/Desktop(或图像发送)”配置分辨率、文件格式、色彩模式及其它设置。
    选择地址簿上的目标地址以便目标地址与以前步骤中的设备界面上的对话框里所显示的“显示名称”相同。
  4. 按下扫描仪上的开始按钮执行扫描。从 从移动设备扫描（远程扫描）的步骤6开始完成剩下操作。

打印

储存在应用程序、相簿、网页以及从邮件接收的文档可以被打印。

打印储存在应用程序中的文档

支持的文件格式为JPEG、TIFF、PNG和PDF（包括压缩的PDF和加密PDF）。
  1. 轻击主界面上的"打印"。
  2. 在iPhone上轻击"选择文件"。
  3. 储存在程序中的文件和文件夹被列出。参考修改文件/文件夹列表实现修改文件/文件夹，参考搜索文件/文件夹实现搜索文件/文件夹。
  4. 选择文件显示带打印设置的预览图。
    关于预览操作，转到预览文件。
  5. 以下打印设置可用。
    打印机
    选择打印机打印并配置打印设置
    图标 表示打印机为默认的打印机。
    份数
    如有必要，轻击以设置打印份数。份数可以设为1-99份。
    色彩模式
    轻击来配置色彩模式。自动，彩色或者灰度可以被选择。
    纸张尺寸
    轻击以配置打印的纸张尺寸。以下尺寸可以被设置：
    A3宽幅, A3, A4, A5, B4, B5, Ledger, Letter, Legal, Executive, 发票, 大页书写纸, 8K, 16K, DL, C5, COM10, Monarch, 日本明信片, 240 x 332, Choukei 3, Japanese You #2, Japanese You #4
    双面设置
    如有需要，轻击以配置单面打印或双面打印。有关双面打印，还可以配置便笺或书籍的装订方向。
    多合一设置
    轻击实现缩小图像尺寸仪适应多页在单页纸上。
    多合一：一合一（一张纸上1页），二合一（一张纸上2页）及四合一（一张纸上4页）可以选择。
    顺序：二合一打印可选择“从左到右”和“从右到左”。四合一打印可以选择“从左上到右”、“从左上到下”、“从右上到左”、“从右上到下”。
    注意：您可以为TIFF，压缩的PDF，黑字增强的PDF和由夏普扫描仪扫描的PDF以外的PDF文件指定多合一。
    打印范围
    设置打印范围打印。
    所有页面：打印机所有页。
    打印页码：指定要打印的第一页页码和最后一页的页码。
    直接输入：输入页码和/或页码范围。对于页码范围，请使用连字符。使用逗号分隔每页或范围。
    注意：您不能为压缩的PDF，黑字增强的PDF和由夏普扫描仪扫描的PDF以外的PDF文件指定打印区域。
    保留
    轻击“保留”实现将打印任务作为文件保存在打印机硬盘中，有需要时可以从打印机操作面板打印任务。设置“仅保留”为“开”启用该功能。保留文件的位置为主文件夹。
    若“指定密码”设置为“开”，则该密码（5-8个数字）确保了储存在打印设备的机密任务和文件不能打印，除非在打印设备操作面板上输入正确的密码。
    注意：您不能保留加密的PDF文件。它可以通过完成步骤7的操作安全地被打印出来。
  6. 轻击"打印到（打印机名称）"（在iPad上轻击"打印"），然后显示对话框"开始打印吗？"。轻击"好"实现打印。
    注意：如果选中的打印机没有合适的选购件来打印文件，出现错误。请选择另一台打印机。
要打印加密的PDF，请操作下面的步骤：
注意：下面的操作以MX-2610N/MX-3110N/MX-3610N为例，打印步骤根据打印机机型有所不同。
  1. 出现对话框"加密的PDF进入缓冲池队列中"，轻击"好"。
  2. 轻击"工作状态"。
  3. 选择"打印"标签，然后轻击"打印缓冲池"。
  4. 从应用程序中发出的打印任务被列出。选择此任务，然后在活动面板上轻击"输入加密PDF密码"。
  5. 显示对话框"请输入密码"。轻击"好"开始打印。

文件管理

储存在应用程序中的文件或者文件夹可以被管理。可用动作包括列出储存在应用程序中文件和文件夹，搜索文件和文件夹，预览文件，重命名文件/文件夹，移动、删除、或者创建文件夹。

修改文件和文件夹列表

您可以搜索文件/文件夹或整理文件/文件夹列表。
  1. 轻击主界面上的“文件管理”按钮。
  2. 储存在应用程序中的文件和文件夹被列出。
  3. 整理按钮和所搜栏显示在文件/文件夹列表中（关于搜索文件/文件夹，请参考搜索文件/文件夹）
    表示当前整理方式和排序的下列图标显示在整理按钮中。
    4. 图标 整理方式和排序
    时间戳递增排序
    时间戳递减排序
    文件名递增排序
    文件名递减排序
    文件大小递增排序
    文件大小递减排序
    文件格式递增排序
    文件格式递减排序
  4. 轻击整理按钮整理。整理方式和排序可以更改。整理方式包含时间戳、文件名、文件大小、和文件格式。轻击实现递增排序，实现递减排序。

搜索文件/文件夹

  1. 轻击主界面上的“文件管理”按钮。
  2. 储存在程序中的文件和文件夹被列出。
  3. 整理按钮和所搜栏显示在文件/文件夹列表中（关于整理文件/文件夹，请参考“修改文件/文件夹列表”）
  4. 请在搜索栏输入您要查找的完整的文件名称或文件名称的一部分。包含搜索字符的文件列表显示。
  5. 从搜索结果中执行进一步搜索，请轻击搜索栏右边的“高级搜索”。
    关于高级搜索结果，可以指定以下条件：
    包含子目录
    搜索中包含子文件夹。
    文件夹
    搜索中包含文件夹。
    PDF
    搜索中包含PDF。
    TIFF
    搜索中包含TIFF。
    图象文件(JPEG,PNG)
    搜索中包含JPEG和PNG。
  6. 您可以更改搜索结果的显示排序。请参考更改文件/文件名列表。

预览文件

  1. 轻击主界面上的“文件管理”按钮。
  2. 储存在程序中的文件和文件夹被列出。
  3. 轻击任意文件预览。
    注意：若移动设备的可用内存不够，则预览不能显示。
    某些文件格式不能预览或旋转（参阅手册样式）。若您要预览，请轻击“用其他应用程序确认”选择一个应用程序预览文件。
  4. 使用两个手指实现缩放预览图像。
    轻击 90度顺时针旋转预览图。
    注意：下面格式的文件可以被旋转。
    • JPEG, TIFF, PNG和通过夏普扫描仪扫描（除加密的PDF外）的PDF。
    注意：即使在预览图旋转和/或缩放后进行了其他操作，图像的初始尺寸和/或方向始终保持。

重命名文件/文件夹

  1. 轻击主界面上的“文件管理”按钮。
  2. 储存在程序中的文件和文件夹被列出。轻击"编辑"按钮。
  3. 复选框出现在列表的左侧，菜单栏出现在底部。
  4. 轻击您要重命名的文件/文件夹的复选框，然后轻击菜单栏上的 。
    注意：一次操作只能更改一个文件/文件夹名称。
  5. 对话框"重命名文件（文件夹）"显示，请输入一个新的名字。
    轻击"保存"实现重命名。
    注意：文件的扩展名不可以被更改。

移动文件/文件夹

  1. 轻击主界面上的“文件管理”按钮。
  2. 储存在程序中的文件和文件夹被列出。轻击"编辑"按钮。
  3. 复选框出现在列表的左侧，菜单栏出现在底部。
  4. 轻击想要移动的文件/文件夹的复选框（多个文件/文件夹可以被选中），然后在菜单栏中轻击。
  5. "移动"对话框显示。选择目标文件夹并且轻击"移动到这里"

删除文件/文件夹

  1. 轻击主界面上的“文件管理”按钮。
  2. 储存在程序中的文件和文件夹被列出。轻击"编辑"按钮。
  3. 复选框出现在列表的左侧，菜单栏出现在底部。
  4. 轻击要删除的文件/文件夹的复选框（多个文件/文件夹可以被选中），然后在菜单栏中轻击。
  5. “删除文件/文件夹”对话框显示。轻击“好”实现删除文件/文件夹。
您也能用以下方式删除文件：
  • 在步骤2，滑动列表中的文件然后轻击"删除"。
  • 轻击预览屏幕底部的"删除文件"。
    注意：此操作过程中文件夹不能被删除。

创建文件夹

  1. 轻击主界面上的“文件管理”按钮。
  2. 储存在程序中的文件和文件夹被列出。
  3. 移动到要创建文件夹的文件夹中，然后轻击"编辑"按钮。
  4. 复选框出现在列表的左侧，菜单栏出现在底部。轻击菜单栏中的 。
  5. "创建文件夹"对话框显示。输入文件夹名称。
    轻击"保存"实现创建文件夹。
