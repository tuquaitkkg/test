Printer/Scanner Settings

Set up printers and scanners to print from and send data to this application.

Find Printers/Scanners

  1. Tap "Settings" on the main screen.
  2. Tap "Printer/Scanner Settings."
  3. Tap "Find Printers/Scanners." Printers and scanners on the network are discovered automatically.
    Note: The printers/scanners must be on the same network segment where your device is connected. If you use a printer/scanner on a different network segment, see     Add Printers/Scanners Manually.
  4. Discovered printers and scanners are displayed in a list.
    "Product Name" and "Location" information retrieved from a printer/scanner is displayed. ("Product Name(Location)" or "Product Name(IP Address)" is displayed above "IP address").
    Note: The display format ("Product Name(Location)" or "Product Name(IP Address)") in the list can be configured in Application Settings. "SHARP" which is included in the "Product Name" retrieved from the printer/scanner is not displayed.

Add Printers/Scanners Manually

  1. Tap "Settings" on the main screen.
  2. Tap "Printer/Scanner Settings."
  3. Tap "Add Manually" if you wish to add any printers/scanners that are not in the same network segment as your device (cannot be added on Find Printers/Scanners).
  4. The following items can be set under "Printer/Scanner Information" and "Behavior Settings".
    Tap "Save" to save the settings. If you wish to cancel the addition, press "Back" button before tapping "Save."
    Name
    Enter printer/scanner name that you wish to add. If a name is not entered, the device's IP address will appear as the name.
    IP Address
    Enter printer/scanner IP address. This is a required field.
    Note: The IP address that has already been added manually cannot be entered.
    Port Number
    Enter a port number for the printer/scanner. This is a required field.
    Port Number is necessary to print from this application. Enter the same port number as configured on the printer. If different port number is entered, this application cannot print to the printer.
    To check the port number in the printer,
    If "Use Raw Print" in Application Settings is unchecked: See [Application Settings]-[Setting for Print from the PC] on the printer's web page. The default port number for both printer and this application is 21.
    If it is checked: See [Network Settings]-[Print Port Settings]-[RAW] tab in the printer’s Web page. The default port number is 9100.
    Set as Default Printer/Scanner
    When selected "I(ON)", the default printer/scanner is set to use this printer/scanner for "Print" or "Scan".
    Note: "Set as Default Printer/Scanner" cannot be configured if "Auto Printer/Scanner Selection" is checked under Application Settings.

View and Edit Printer/Scanner Information

  1. Tap "Settings" on the main screen.
  2. Tap "Printer/Scanner Settings."
  3. Registered printers and scanners are displayed as list.
    The icon indicates automatically added printers/scanners.
    The icon indicates manually added printers/scanners.
  4. Tap a printer/scanner to view its information.
  5. The information about the selected printer/scanner is displayed. The items displayed here are available in Add Printers/Scanners Manually.
    Note: For automatically added printers/scanners, IP address is not editable.
    For manually added printers/scanners, Product Name and Location is not editable. Product Name and Location fields are auto-filled when the application establishes an initial contact with the printer/scanner.
  6. When "scan from mobile device" is available, the following additional settings are available under "Operation Settings".
    Scan from your mobile device
    When enabled (select "I(ON)"), configure scan settings and scan on your mobile device.
    Display Name
    Display Name will be displayed along with Verification Code on both scanner operation panel and your mobile device to ensure your target scanner.
    Automatically generate verification codes
    When enabled (select "I(ON)"), automatically generate a Verification Code. If it is disabled (select "O(OFF)"), you need to enter a code in the Verification Code field.
    Verification Code
    Verification Code will be displayed along with Display Name on both scanner operation panel and your mobile device to ensure your target scanner.
    Reset scan parameters
    Tap this to reset scan settings configured for the last scan operation.
  7. Tap "Save" to save the settings.

Move or Delete Printer/Scanner in the List

  1. Tap "Settings" on the main screen.
  2. Tap "Printer/Scanner Settings."
  3. Registered printers and scanners are displayed as a list.
  4. Tap "Edit".
  5. Tap , then tap "Delete" if you wish to delete a printer/scanner from the list.
  6. If you wish to rearrange the order of the list, tap and hold , then drag it to the desired location.
    Note: Deleted printers/scanners can be added again with     Find Printers/Scanners or Add Printers/Scanners Manually.

User Settings

Set up the user information.
  1. Tap "Settings" on the main screen.
  2. Tap "User Settings."
  3. "Profile Information" is the setting required for registering destination to a scanner on "Scan" operation. The following settings are available:
    Display Name
    Displayed in the scanner address book as a scan destination. The default value is same as the value of "Name" configured in User Information in the device's settings.
    Initial
    Initial is used for a sort key on displaying "Display Name" in the address book screen of the scanner.
  4. "User Authentication" is the setting required for printing to a printer on which user authentication is enabled. The following settings are available:
    Use Login Name for authentication
    Select this when Login Name is used for user authentication on the printers. Type in "Login Name" and "Password".
    Login Name
    Enter your login name used for user authentication on the printers.
    Password
    Enter your password used for user authentication on the printers.
    Use User Number for authentication
    Select this when User Number is used for user authentication on the printers in your network. Type in "User Number".
    User Number
    Enter your user number used for user authentication on the printers.
  5. When "Default Job ID" is selected, printers display "User Name" and "Job Name" configured here on the operation panel.
    User Name
    Enter a user name which will be displayed on printer operation panel. If this is not set, the printer uses the value, "SharpdeskM User".
    Job Name
    Enter a job name which will be displayed on printer operation panel. If this is not set, the printer uses the name of the file to be printed.
  6. Tap "Save" to save the settings.

Application Settings

Set up the operation of the application.
  1. Tap "Settings" on the main screen.
  2. Tap "Application Settings."
  3. The following settings can be set in "Operation Settings."
    Profile Automatic Delete
    Select "I(ON)" to delete the destination on the scanner after scan is completed.
    Auto Profile Update
    Select "I(ON)" to overwrite the destination that was registered at last scanning without confirmation.
    Save Imported Files
    Select "I(ON)" to keep a file received from other application for printing from this application.
    Auto Printer/Scanner Selection
    Select "I(ON)" to automatically set a default printer/scanner in the connected wireless network (SSID).
    High Quality Print
    Select "I(ON)" for higher quality printing.
    Use Raw Print
    Select "I(ON)" to use Raw port (default port number is 9100) instead of FTP port (default port number is 21) for printing.
    Default Name for Automatically Added Printer/Scanner
    Display format can be set for the printers/scanners that are automatically added. "Product Name(Location)" or "Product Name(IP Address)" can be selected.
  4. "SNMP Settings" is required to set if the network to which your device is connected uses other than default SNMP GET Community string. Select "O(OFF)" in "Public Get Community" if you use this application in such a network, and enter the strings that are used in the network in "Enter Community String(s)." Up to 10 strings can be entered. For further assistance, please ask your network administrator about SNMP GET Community string.
  5. Tap "Save" to save the settings.

Scan

This function receives and stores scanned documents.
  1. Tap "Scan" on the main screen.
  2. Tap "Scanner: (scanner name)" (on iPad, tap the button under "Scanner") to select the desired scanning device.
    This icon identifies the scanner assigned as the default scanner.
Operations may vary depending on scanner.
Please go to Scan from Your Mobile Device to learn how to execute a scan from your mobile device. If scan setting option is not available on your mobile device, please go to Scan from the Scanner Operation Panel.

Scan from Your Mobile Device

Please complete the following steps to configure scan settings on your mobile device. Go to the application's Web site for a list of supported scanners.
All operations except "setting a document on the scanner" shall be completed within the application.
Note: Please confirm scanner home screen or user authentication screen is displayed on the scanner. When external authentication is enabled, user authentication screen will be displayed. Upon successful authentication, scanner home screen will be displayed.
  1. If you selected the scanner that supports Scan from Your Mobile Device feature, the setting menus and current scan settings will be displayed.
    Note: When you disable this feature, you can initiate scan from scanner. Please go to Scan from the Scanner Operation Panel for further instructions.
    The following settings are available. If you are using iPhone, settings can be stored by tapping "Complete". If you are using iPad, settings are automatically stored.
    Color Mode
    Settings to scan color originals (Auto/Full Color/Grayscale/Mono2). The default value is "Auto".
    Note: When "Long Size" is selected for "Original Size" under "Original", only "Mono2" is available.
    Original
    Settings for original document to be scanned. The following items can be set:
    Original Size
    While "Auto" is set as default, non-standard original paper size cannot be detected by the scanner. You need to manually change the original size setting.
    Detected paper size (display only)
    When you set a document on the target scanner, the original paper size will be automatically detected and displayed on your mobile device and on the scanner operation panel.
    Store Size
    The size of the scanned image can be manually set. If this size is different than the detected paper size, the image will be automatically adjusted to fit the page. The default value is "Auto".
    Note: You can only select "Auto" when any of the following original paper sizes is selected: Long Size/Japanese Postcard/Business Card/2L Size/Card/Custom Size.
    Custom Size Registration
    Custom original paper sizes can be registered. Up to 5 custom sizes can be registered.
    Image Orientation
    Select orientation of the image. You can select either "Rotation Off" or "Rotation 90 Degrees". The default value is "Rotation Off".
    2-Sided Copy
    Select 1-sided or 2-sided original.
    1-Sided, 2-Sided(Book) or 2-Sided(Tablet) can be selected. The default value is "1-Sided".
    File Format
    Scan file format (file type, compresson mode, and compression ratio) can be set. The following items can be set.
    "Color/Grayscale"/"B/W"
    Format mode for scanning (Color/Grayscale, or B/W) can be selected.
    File Type
    Scan file type can be selected.
    When "Color/Grayscale" is selected in the above setting, PDF, PDF/A, TIFF or JPEG file types are available.
    When "B/W (monochrome)" is selected in "Color", PDF, PDF/A, and TIFF file types are available.
    Compact PDF
    The type of Compact PDF (None, Compact or Compact Ultra Fine) can be selected when "Color/Grayscale" is selected in the "Color/Grayscale"/"B/W" setting, and "PDF" or "PDF/A" is selected in "File Type".
    Note: Compact PDF is available only if the Scanner Expansion Kit is installed in the scanner.
    Comp. Ratio
    Compression Ratio (low, middle, high) can be selected when "Color/Grayscale" is selected in the "Color/Grayscale"/"B/W" setting. In addition, "Bk Letter Emphasis" can be selected if the scanner supports Bk letter emphasis PDF and "PDF" is set as the "File Type".
    Compression Mode
    Compression Mode (none, MH(G3), MMR(G4)) can be selected when "B/W" is selected in "Color".
    Specified Pages per File
    This feature is available when you select "PDF", "PDF/A", or "TIFF" file type. When selected, scanned document is split into files by specified number of pages. (E.g. when 2 pages is specified, the scanned document is grouped by 2 pages to create multiple files.)
    Encryption
    When "PDF" file type is selected, encrypted PDF option is available. When checked, a "Password" should be assigned to open the encrypted PDF.
    Resolution
    Scan resolution can be set. The following resolution options are available.
    100x100dpi, 150x150dpi, 200x200dpi, 300x300dpi, 400x400dpi, 600x600dpi
    Note: 150x150dpi can be selected only on the scanners that support the resolution.
    Note: Only 300x300dpi is available when you select "Compact" or "Compact Ultra Fine" under "Compact PDF".
    Other Settings
    Additional scan settings are available under "Others". The following items are available:
    Exposure
    Exposure (Auto, Characters, Text/Prtd.Photo, Printed Photo, Text/Photo, Photo, Map) for scanning can be selected.
    Exposure Level
    Exposure level (1 to 5) for scanning can be selected. This can be set if "Auto" is selected for "Exposure".
    Blank Page Skip
    If set to other than "Off", if the original contains blank pages or back shadow, the scanner will automatically detect them and skip them when scanning. Setting options are "Off", "Skip Blank Page", and "Skip Blank and Back Shadow".
  2. When you complete all settings, tap "Scan from (scanner name)" (on iPad, tap "Scan") to start scanning the document.
  3. After tapping "Scan" button, the information of the target scanner ("Display Name" and "Verification Code") will be displayed on the scanner operation panel and move to a waiting (scanning ready) stage.
  4. Confirm the device information and set document on the scanner.
  5. Select “Scan” on a pop-up dialogue to continue.
  6. Scanned data is sent to this application and the data is stored into the memory.
    If a single data is received, the preview of the data is displayed.
    For preview operation, go to Preview a File.
  7. Tap "Save this Image".
  8. Scanned data, already named at the scanner, can be renamed at this stage.
    If desired, enter a new file name and then tap "Save".
  9. Select the destination folder and tap "Save Here" at the bottom of the screen to store this data as a file into the folder.
If multiple scanned data is received, please complete the following steps:
  • The received data is listed. Tap "Bulk Save" and then complete steps 7 and 8 to store the files into the selected folder in the device. (A sequential number is added to the end of the file name entered in step 7.)
    Note: You may tap any data in the list to preview. However, this data can only be stored together with the rest of the received data.

Scan from the Scanner Operation Panel

If “Scan from your mobile device” is not available on the scanner, use following steps to scan from scanner operation panel.
Resolution, file format and other scan settings must be configured on the MFP operation panel when scanning. (see step 3)
Note: Please be sure address book is not displayed on the scanner operation panel. If address book is displayed, this operation will not work.
  1. Tap "Scan from (scanner name)" (on iPad, tap "Scan").
  2. Tap "OK" when the dialog "Do you want to Scan?" is displayed. The dialog with "Display Name" and "Initial" is displayed indication the application is in the waiting state.
  3. Set a document on the scanner and select "FTP/Desktop (or IMAGE SEND)" to configure resolution, file format, color mode or other settings.
    Select the destination on the address book so that the destination is the same as "Display Name" displayed in the dialog of the device's screen in the previous step.
  4. Push Start button on the scanner to execute scanning. Begin at step 6 in Scan from Your Mobile Device(Remote Scan) to complete the process.

Print

The documents stored in this application, photo images, Web pages and received e-mail can be printed.

Print a Stored Document in this Application

Supported file formats are JPEG, TIFF, PNG and PDF (including compact PDF and encrypted PDF).
  1. Tap "Print" on the main screen.
  2. On iPhone, tap "Select File".
  3. Files and folders stored in this application are listed. See Modify Files/Folders List to Modify files/folders list or Search Files/Folders to search files/folders.
  4. Select a file to display a preview with print settings.
    For preview operation, see Preview a File.
  5. The following print settings are available:
    Printer
    Select the printer to print and configure print settings.
    This icon identifies the printer assigned as the default printer.
    Quantity
    Tap to configure print quantity if necessary. Quantity can be configured from 1 set to 99 sets.
    Color Mode
    Tap to configure color mode. Either Auto, Color or Grayscale can be selected.
    Paper Size
    Tap to configure paper size to be printed. The following sizes can be configured:
    A3 Wide, A3, A4, A5, B4, B5, Ledger, Letter, Legal, Executive, Invoice, Foolscap, 8K, 16K, DL, C5, COM10, Monarch, Japanese Postcard, 240 x 332, Choukei 3, Japanese You #2, Japanese You #4
    Duplex Settings
    Tap to configure 1-sided print or 2-sided print if necessary. Regarding 2-sided print, it can be configured along with binding direction of Tablet or Book.
    N-Up Settings
    Tap to reduce image size and fit multiple pages on a single page.
    N-Up: 1-Up(1 page per a single sheet), 2-Up(2 pages per a single sheet) or 4-Up(4 pages per a single sheet) can be selected.
    Order: For 2-Up printing select "Left to Right" or "Right to Left". For 4-Up printing select "Right, and Down", "Down, and Right", "Left, and Down" or "Down, and Left".
    Note: You cannot specify N-Up for TIFF, compact PDF, Bk letter emphasis PDF and PDFs other than those scanned by Sharp scanners.
    Print Area
    Configure print range to print.
    All Pages: Print all pages.
    Print Pages: Specify the number of the first and last pages that you want ot print.
    Direct Input: Enter page numbers and/or page ranges. For page ranges, use a hyphen. You can separate each page or range with a comma.
    Note: You cannot specify print area for compact PDF, Bk letter emphasis PDF and PDFs other than those scanned by Sharp scanners.
    Retention
    Tap "Retention" to hold a print job as a file on the printer's hard disk, allowing the job to be printed from the printer operation panel when needed. Set "Hold Only" to "ON" to enable this feature. Main folder is specified as the location of the held file.
    If "Specify Password" is set to "ON", the password (from 5 to 8 digit number) allows a confidential job to be stored on the printing device and the file cannot be printed unless the correct password is entered at the printing device's operation panel.
    Note: You cannot specify retention for Encrypted PDF. It can be securely printed by completing the operations from step 7.
  6. Tap "Print to (printer name)" (on iPad, tap "Print"), then the dialog "Do you want to Print?" is displayed. Tap "OK" to print.
    Note: If selected printer does not have appropriate options to print the file, an error message is displayed. Please select another printer.
To print an encrypted PDF, complete the following steps:
Note: The following operation is an example using MX-2610N/MX-3110N/MX-3610N, and the steps for print may vary per printer model.
  1. The dialog "The encrypted PDF data is in the spool queue." is displayed. Tap "OK".
  2. Tap "Job Status".
  3. Select "Print" tab, and then tap "Spool".
  4. The print jobs sent from this application are listed. Select the job, and then tap "Enter Password for Encrypted PDF" in Action Panel.
  5. The dialog "Please enter Password." is displayed. Tap "OK" to start printing.

Manage Files

Files or folders stored in the application can be managed. Available actions include listing files and folders stored in the application, searching for files and folders, previewing files, changing a file/folder name, move, delete, or create a folder.

Modify Files/Folders List

You can search files/folders or sort the files/folders list.
  1. Tap "Manage Files" on the main screen.
  2. Files and folders stored in this application are listed.
  3. Sort button and search bar are displayed in the files/folders list (for searching files/folders, see Search Files/Folders).
    One of the following icons expressing current sort condition and order is indicated in the Sort button.
    4. Icon Sort condition and order
    Ascending order of time stamp
    Descending order of time stamp
    Ascending order of file name
    Descending order of file name
    Ascending order of file size
    Descending order of file size
    Ascending order of file format
    Descending order of file format
  4. Tap the sort button to sort. Sort condition and order can be changed. Sort conditions include time stamp, file name, file size and file format. Tap for ascending order and for descending order.

Search Files/Folders

  1. Tap "Manage Files" on the main screen.
  2. Files and folders stored in this application are listed.
  3. Sort button and search bar are displayed in the files/folders list (for sorting files/folders, see "Modify Files/Folders List").
  4. Enter the entire file name, or a portion of the file name that you would like to find into the search bar. A list of files containing the searched-for characters will be displayed.
  5. Further search from the search result can be conducted with tapping "Advanced Search" to the right of the search bar.
    The following conditions can be specified for advanced search result.
    Include subfolders
    Include subfolders in search.
    Folder
    Include folders in search.
    PDF
    Include PDF in search.
    TIFF
    Include TIFF in search.
    Image File(JPEG,PNG)
    Include JPEG and PNG in search.
  6. You can change the display order of the search results. See Modify Files/Folders List.

Preview a File

  1. Tap "Manage Files" on the main screen.
  2. Files and folders stored in this application are listed.
  3. Tap any file to preview.
    Note: Preview is not displayed if free memory in the device is not sufficient.
    Some file formats cannot be previewed or rotated (see the Operation Manual). If you wish to preview, tap "Check with other application" to select an application to preview the file.
  4. Use two fingers to zoom in/out the preview image.
    Tap to rotate the preview 90 degrees clockwise.
    Note: The files with the following formats can be rotated:
    • JPEG, TIFF, PNG, and PDF scanned by Sharp scanners(except encrypted PDFs)
    Note: Original size and/or orientation of the image is kept even if another operation is performed after the preview was rotated and/or zoomed.

Rename a File/Folder

  1. Tap "Manage Files" on the main screen.
  2. Files and folders stored in this application are listed. Tap "Edit" button.
  3. Check boxes appear in the left of the list and the menu bar appears in the bottom.
  4. Tap the check box of the file/folder you wish to rename, then tap in the menu bar.
    Note: You can rename only 1 file/folder at one operation.
  5. The dialog "Change File(Folder) Name" appears. Enter a new name.
    Tap "Save" to rename.
    Note: The extension name of the file cannot be changed.

Move Files/Folders

  1. Tap "Manage Files" on the main screen.
  2. Files and folders stored in this application are listed. Tap "Edit" button.
  3. Check boxes appear in the left of the list and the menu bar appears in the bottom.
  4. Tap the check box of the file/folder you wish to move (multiple files/folders can be checked) and tap in the menu bar.
  5. "Move" dialog is displayed. Select the destination folder and tap "Move Here."

Delete Files/Folders

  1. Tap "Manage Files" on the main screen.
  2. Files and folders stored in this application are listed. Tap "Edit" button.
  3. Check boxes appear in the left of the list and the menu bar appears in the bottom.
  4. Tap the check box of the file/folder you wish to delete (multiple files/folders can be checked) and tap in the menu bar.
  5. "Delete file(s)/folder(s)" dialog is displayed. Tap "OK" to delete the files/folders.
You can also delete a file in the following ways:
  • In step 2, swipe the file in the list and tap "Delete".
  • Tap "Delete this File" at the bottom of the preview screen.
    Note: Folders cannot be deleted in this operation.

Create a Folder

  1. Tap "Manage Files" on the main screen.
  2. Files and folders stored in this application are listed.
  3. Move to the folder under which you wish to create a new folder, then tap "Edit" button.
  4. Check boxes appear in the left of the list and the menu bar appears in the bottom. Tap in the menu bar.
  5. "Create Folder" dialog is displayed. Enter a new folder name.
    Tap "Save" to create the folder.

Print from External Application

Data sent from other applications in your device can be printed from this application.
  1. Open the file that you wish to print (JPEG, TIFF, PNG or PDF) in other application.
  2. Operate "send the file to another application" on that application. (see the operation manual of the application for details).
    "Sharpdesk M" is listed as one of the applications that can receive this file. Then tap "Sharpdesk M".
    Note: If you use this application on iOS 4.3 through 5, up to 9 applications are listed. If "Sharpdesk M" is not listed, please uninstall unnecessary applications.
  3. The rest of the operation is the same as step 5 and after Print a Stored Document in this Application.
